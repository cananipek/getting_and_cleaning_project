# Code Book

The Code Book summarizes the variables in the tidyData.csv file

## Identifiers

1. Subject: subject's id
2. Activity: type of activity performed by the subject. The levels are
	- WALKING(1)
	- WALKING_UPSTAIRS(2)
	- WALKING_DOWNSTAIRS(3)
	- SITTING (4)
	- STANDING (5)
	- LAYING (6)
